## Core argument
Customer segmentation derived from behavioural/demographic data can be directly integrated with AI sales prediction models, improving forecast accuracy by providing demand-side context that transactional data alone cannot capture.

## Relevance to thesis
- **SRQ3 primary reference**: provides the academic grounding for using Indeks Danmark consumer segments as an enrichment signal for Nielsen sales forecasts
- Demonstrates that customer profile → sales prediction pipeline is methodologically sound and empirically validated
- Justifies the Synthesis Agent's consumer context adjustment step

## Gap / limitation
Direct marketing context — not retail panel data. Does not address multi-agent orchestration or computational constraints.
